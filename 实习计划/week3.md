1.爬楼梯：
假设你正在爬楼梯。需要 n 阶你才能到达楼顶。
每次你可以爬 1 或 2 个台阶。你有多少种不同的方法可以爬到楼顶呢？

    斐波拉契数列，代码相当简单，一开始考虑了递归的做法，但是超时了，问题出在递归会大量重复调用，消耗大量性能，所以之后想了用数组，用哪个拿哪个就行了。

    我的做法，空间复杂度O(n)：
        dp = [0, 1, 2]          
        i = 3                   
        while i <= n:
            dp.append(dp[i-1] + dp[i-2])
            i += 1
        return dp[n]

    不浪费空间的做法,空间复杂度O(1)：
        if n <= 2:
            return n
        a, b = 1, 2  # 分别代表 f(i-2), f(i-1)
        for _ in range(3, n + 1):
            a, b = b, a + b
        return b
