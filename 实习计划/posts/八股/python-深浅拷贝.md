## 一句话定义

浅拷贝是创建并复制了原本的外层容器，而内部元素仍然指向原本的内存位置；深拷贝是递归地复制了外层容器和内部元素以及内部的所有嵌套对象，并且全部开辟新的内存空间。

## 我的理解

浅拷贝就是对外层容器的拷贝，但内部仍然引用旧的元素。深拷贝就是对“整个”对象的复制，它会递归地复制外层和所有内层嵌套对象和元素，然后为整个复制体开辟新的内存空间。浅拷贝的应用一般在那些非嵌套对象，或者嵌套但是内部的东西长期不变的对象。深拷贝一般用在嵌套的对象，并且内部信息需要每个实例独自更新的对象，不过深拷贝一般运行开销会很大，要慎用。

## 例子
```python
#浅拷贝
import copy
a = [[1, 2], [3, 4]]
b = copy.copy(a)      # 浅拷贝:外层新列表,内层还是 a 的引用

b[0][0] = 99          # 改 b 的内层
print(a)              # [[99, 2], [3, 4]]  ← a 也被改了,因为内层是同一个对象

b[0] = [9, 9]         # 改 b 的外层
print(a)              # [[99, 2], [3, 4]]  ← a 不受影响,外层是独立的


#深拷贝
import copy
a = [[1, 2], [3, 4]]
b = copy.deepcopy(a)  # 深拷贝:递归把每一层都新建

b[0][0] = 99          # 怎么改都不影响 a
print(a)              # [[1, 2], [3, 4]]  ← 完全独立
