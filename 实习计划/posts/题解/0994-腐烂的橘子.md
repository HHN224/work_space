## 题意

给定一个二维数组，0代表空，1代表新鲜橘子，2代表腐烂橘子，每分钟，腐烂橘子四个方向上的橘子都会腐烂，求橘子全部腐烂的分钟数，如果不可能，返回-1

## 思路

基本结构是一个while循环包含两个for循环的结构，最关键的是has_rot参数，每次while循环开始都把他设置成False，第一个for循环：每当腐烂橘子影响到新鲜橘子，就把has_rot设置成True，然后把新鲜橘子暂时标记为3——意思是等会统一推进时间并将其变为腐烂状态。第二个for循环：统一把3变成2，然后时间+1。因为有可能有新鲜的橘子在完全的孤岛，永远不会被感染，所以在外层再加一个判断。

## 代码

```python
class Solution(object):
    def orangesRotting(self, grid):
        """
        :type grid: List[List[int]]
        :rtype: int
        """
        
        rows = len(grid)
        if rows == 0:
            return 0
        cols = len(grid[0])
        dirs = [(1,0), (-1,0), (0,1), (0,-1)]
        minutes = 0

        while True:
            has_rot = False
            # 第一步：标记这一轮要腐烂的橘子为3，不实时修改2扩散
            for i in range(rows):
                for j in range(cols):
                    if grid[i][j] == 2:
                        for dr, dc in dirs:
                            ni, nj = i + dr, j + dc
                            if 0 <= ni < rows and 0 <= nj < cols and grid[ni][nj] == 1:
                                grid[ni][nj] = 3
                                has_rot = True
            # 本轮没有橘子腐烂，扩散结束
            if not has_rot:
                break
            # 第二步：把标记3全部转为腐烂2
            for i in range(rows):
                for j in range(cols):
                    if grid[i][j] == 3:
                        grid[i][j] = 2
            minutes += 1
        
        # 检查是否还有新鲜橘子残留
        for i in range(rows):
            for j in range(cols):
                if grid[i][j] == 1:
                    return -1
        return minutes
```

## 复盘

- 时间 O(K×M×N)：K为while循环轮数，空间 O(1)
