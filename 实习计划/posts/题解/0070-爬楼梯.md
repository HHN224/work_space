## 题意

有n阶楼梯要爬，每次可以选爬1阶或爬2阶，求最终到顶时有多少种爬法

## 思路

这题是动态规划，关键在于理清每一阶和其他阶的关系。假设现在爬到第n阶段，那么爬到第n阶有两种可能：第一种是在n-1阶爬一格；第二种是在n-2阶爬两格。所以dp[n] = dp[n-1] + dp[n-2]

## 代码

```python
class Solution(object):
    def climbStairs(self, n):
        """
        :type n: int
        :rtype: int
        """
        
        list = []
        list.append(1)
        list.append(1)
        list.append(2)
        i = 2
        while i <= n:
            list.append(list[i] + list[i-1])
            i += 1
        return list[n]
```

## 复盘

- 时间 O(n)，空间 O(n)
