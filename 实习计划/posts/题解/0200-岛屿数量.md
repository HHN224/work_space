## 题意

给定一个二维网格，其中‘1’代表陆地，‘0’代表水，斜着不算岛屿有连接，计算其中的岛屿数量。

## 思路

递归DFS：这是比较容易想到的方案，利用递归实现从岛屿任意位置清除岛屿，然后每次遇到一个陆地就清空一次，并且计数加一，最后返回计数值即可。

迭代DFS：也是比较容易想到的方案，比递归DFS大概快了100ms左右，使用栈的风格，发现1之后给他改0然后把四个方向上的元素等于1的变0，并且让周围的入栈，栈里只要有东西就都接着运行，这样就可以类似递归地将单个岛屿全清除，并且优势是应用面更广，不用担心递归层数受python默认1000层的限制影响。

```python
#递归DFS
class Solution(object):
    def numIslands(self, grid):
        """
        :type grid: List[List[str]]
        :rtype: int
        """
        def widespread(i,j):
            direct = [[0,1],[0,-1],[1,0],[-1,0]]
            if i >= x or j >= y or i < 0 or j < 0:
                return
            else: 
                if grid[j][i] == '0':
                    return
                grid[j][i] = '0'
                n = 0
                while n < 4:
                    i_move = direct[n][0]
                    j_move = direct[n][1]
                    ni, nj = i+i_move, j+j_move
                    widespread(ni, nj)
                    n += 1
            return

        y = len(grid)
        x = len(grid[0])
        count = 0
        for i in range(x):
            for j in range(y):
                if grid[j][i] == '1':
                    count += 1
                    widespread(i,j)
        return count
```

```python
#迭代DFS
class Solution(object):
    def numIslands(self, grid):
        """
        :type grid: List[List[str]]
        :rtype: int
        """
        
        if not grid or not grid[0]:
            return 0
        rows = len(grid)
        colums = len(grid[0])
        count = 0
        dirs = [(1,0),(-1,0),(0,1),(0,-1)]
        for j in range(rows):
            for i in range(colums):
                if grid[j][i] == '1':
                    stack = [(j,i)]
                    grid[j][i] = '0'
                    count += 1
                    while stack:
                        r, c = stack.pop()
                        for dr, dc in dirs:
                            nr, nc = r+dr, c+dc
                            if 0 <= nr < rows and 0 <= nc < colums and grid[nr][nc] == '1':
                                grid[nr][nc] = '0'
                                stack.append((nr,nc))
        return count
```

- 迭代DFS：时间 O(MN)，空间 O(MN)
