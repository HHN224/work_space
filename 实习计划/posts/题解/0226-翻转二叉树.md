## 题意

给定一个二叉树，要求它的整体翻转

## 思路

DFS：深度优先的方法使用了简单的遍历，边缘条件判断也相当简单，具体可以参考206翻转链表

BFS：广度优先使用了队列的方法进行层序遍历，每次都取出队列的目标节点，将他们的左右子节点互换，然后把左右子节点都入队（如果有），如果队列里所有节点都处理完，则退出。

## 代码

```python
#DFS
class Solution(object):
    def invertTree(self, root):
        """
        :type root: Optional[TreeNode]
        :rtype: Optional[TreeNode]
        """
        if not root:
            return root

        root.left = self.invertTree(root.left)
        root.right = self.invertTree(root.right)
        root.left, root.right = root.right, root.left
        return root
```

```python
#BFS
class Solution(object):
    def invertTree(self, root):
        """
        :type root: Optional[TreeNode]
        :rtype: Optional[TreeNode]
        """
        if not root:
            return root
        queue = [root]
        while queue:
            node = queue.pop(0)
            node.left, node.right = node.right, node.left
            if node.left:
                queue.append(node.left)
            if node.right:
                queue.append(node.right)
        return root
```

## 复盘

- DPS：时间 O(n)，空间 O(h)
- BFS：时间 O(n²)，空间 O(w)
- 可迁移：深度优先的部分和广度优先的部分都相当实用，在二叉树的题目上可以优先思考这两种实现方式
