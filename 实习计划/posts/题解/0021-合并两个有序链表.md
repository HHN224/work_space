## 题意

两个升序链表合并成一个升序链表并返回

## 思路

很基础的链表合并题，最主要是要有一个哨兵保管答案的头节点，然后另一个可移动的指针去处理后面的逻辑，比较关键的位置是如果最后有剩下的部分，要全部合并到答案里。

## 代码

```python
class Solution(object):
    def mergeTwoLists(self, list1, list2):
        """
        :type list1: Optional[ListNode]
        :type list2: Optional[ListNode]
        :rtype: Optional[ListNode]
        """

        dummy = ListNode()
        cur = dummy
        while list1 and list2:
            if list1.val <= list2.val:
                cur.next = list1
                list1 = list1.next
            else:
                cur.next = list2
                list2 = list2.next
            cur = cur.next
        cur.next = list1 if list1 else list2
        return  dummy.next
```

## 复盘

- 时间 O(n)，空间 O(1)
- 可迁移：链表的基础题
