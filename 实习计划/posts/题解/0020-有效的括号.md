## 题意

给定一个字符串，判断里面的所有括号是否都能成对

## 思路

这里思路一开始想到栈，只要确保进来的和顶部对上就可以消除

## 代码

```python
class Solution(object):
    def isValid(self, s):
        """
        :type s: str
        :rtype: bool
        """

        stack = []
        for i in range(len(s)):
            entering = s[i]
            if len(stack) == 0:
                stack.append(entering)
                continue
            up = stack[-1]
            if entering == ")" and up == "(" or entering == "]" and up == "[" or entering == "}" and up == "{":
                stack.pop()
            else:
                stack.append(entering)
        is_empty = (len(stack) == 0)
        if is_empty:
            return True
        else:
            return False
```

## 复盘

- 时间 O(n²)，空间 O(n)
- 优化方向：把括号的对应关系单独存在哈希表中，这样会使逻辑更完美，并且减少冗余变量
