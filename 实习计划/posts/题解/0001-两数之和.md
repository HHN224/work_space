## 题意

对给定的数组nums和target，求nums中两个数的和等于target的所有情况

## 思路

首先想的是穷举，双循环嵌套就可以暴力求出符合target的解。但是时间复杂度O(n^2)显然不理想。

解决思路：双层循环时间复杂度太高，那就换成单层循环，但此时要拿到第二个数则需要新的数据结构，最佳方法是结合哈希表。利用哈希表搜索O(1)的特性，我们一边遍历一边填充哈希表，并且求出每一个数字”需要的值“，这个“需要的值”是target和这个数字本身的差值，然后查找“需要的值”，如果找到，就返回答案。

## 代码

```python
class Solution(object):
    def twoSum(self, nums, target):
        """
        :type nums: List[int]
        :type target: int
        :rtype: List[int]
        """

        hashmap = {}
        for idx, val in enumerate(nums):
                need = target - val
                if need in hashmap:
                    return [idx, hashmap[need]]
                hashmap[val] = idx
```

## 复盘

- 时间 O(n) 空间O(n)
