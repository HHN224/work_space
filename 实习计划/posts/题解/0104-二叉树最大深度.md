## 题意

给定一个二叉树，求此树的根到深的叶节点的距离，即求最大深度

## 思路

对于树的题，可以用DFS(深度优先)，BFS(广度优先)，一般按实际情况选择，这里求深度DFS相当简洁并且快，所以使用DFS。

这里可以直接用递归来实现最简单的DFS，考虑极限条件：递归到某个叶子节点的外面，那么此时应该返回层数0，使得叶子节点层数为1，即可顺利递归。

## 代码

```python
class Solution(object):
    def maxDepth(self, root):
        """
        :type root: Optional[TreeNode]
        :rtype: int
        """

        if not root:
            return 0

        left_depth = self.maxDepth(root.left)
        right_depth = self.maxDepth(root.right)
        return max(left_depth, right_depth) + 1
```

## 复盘（可选）

- 时间 O(n)，空间 O(n)
- 可迁移：对于简答的树的搜索都可以考虑使用这一套递归模式
