## 题意

给定一个整数数组 `nums`，找到一个具有最大和的连续子数组（子数组最少包含一个元素），返回其最大和。

## 思路

暴力解法是枚举所有子数组求和，时间复杂度 O(n²)，显然不可行。

解决思路（Kadane 算法 / 动态规划）：遍历数组时维护一个「当前子数组和」`cur_sum`。核心观察是：如果当前累积和已经变成负数，那么它对后续元素只会“拖后腿”，不如从当前元素重新开始计数。同时用一个变量 `max_sum` 记录遍历过程中遇到的最大值即可。

另一种理解方式：定义 `dp[i]` 为以第 `i` 个元素结尾的连续子数组的最大和，则状态转移方程为 `dp[i] = max(nums[i], dp[i-1] + nums[i])`，最终答案就是 `max(dp)`。由于 `dp[i]` 只依赖 `dp[i-1]`，可以压缩成 O(1) 空间。

## 代码

```python
class Solution(object):
    def maxSubArray(self, nums):
        """
        :type nums: List[int]
        :rtype: int
        """
        length = len(nums)
        cur = s = nums[0]
        for i in range(1, length):
            cur = max(nums[i], cur+nums[i])
            s = max(cur, s)
        return s
```
