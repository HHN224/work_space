## 题意

给定一个链表，判断是否有环

## 思路

在头部放出两个指针，一个速度是1，另一个速度是2，一直让他们前进，那么如果有环的话速度2的就会撞上速度1的，如果没环就当速度2撞到“next为空”的节点就退出。

## 代码

```python
class Solution(object):
    def hasCycle(self, head):
        """
        :type head: ListNode
        :rtype: bool
        """
        fast = slow = head
        while fast and fast.next:
            slow = slow.next
            fast = fast.next.next
            if slow == fast:
                return True
        return False
```
