## 题意

给定一个链表，返回一个方向完全相反的链表

## 思路

第一种思路：正常写，制造三个指针，不断向右推进他们，每过一个就把链表两个节点间的指向反向，这种解法有很多实现方法，这里贴出来一种行数最少的，有点类似初学c语言的时候写的两个变量交换的程序；

第二种思路：使用递归的思路，递归要考虑的是极限条件，也就是当程序递归到最下层的时候会发生什么，会返回什么。这里考虑的话，也就是返回一个尾节点，那么显然，这个尾节点就是头节点，所以把他=new_head。然后和第一种思路一样，把两个节点间的箭头反向，最后一路返回new_head即可。

## 代码

```python
class Solution(object):
    def reverseList(self, head):
        """
        :type head: Optional[ListNode]
        :rtype: Optional[ListNode]
        """
        
        temp = ListNode()
        pre = None
        cur = head
        while cur:
            temp = cur.next
            cur.next = pre
            pre = cur
            cur = temp
        return pre
```

```python
class Solution(object):
    def reverseList(self, head):
        """
        :type head: Optional[ListNode]
        :rtype: Optional[ListNode]
        """
        if not head or not head.next:
            return head
        new_head = self.reverseList(head.next)
        head.next.next = head
        head.next = None
        return new_head
```
