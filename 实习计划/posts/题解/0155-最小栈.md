## 题意

要设计一个最小栈，要求：实现pop删除最顶数据，push从顶端加入数据，top获取顶端数据，getMin获取栈中最小值

## 思路

这题一开始认为可以普通地实现一个栈，然后调用getMin的时候每次都遍历一遍然后找到最小值，但是最终超时。

解决思路：在这个对象内再维护一个最小值的栈，当入栈数值比最小值小，最小值的栈就也入栈，出栈类似。这样当我们取最小值的时候就取最小值的栈栈顶即可。

## 代码

```python
class MinStack(object):

    def __init__(self):
        self.data = []
        self.min = []
        

    def push(self, val):
        """
        :type val: int
        :rtype: None
        """
        self.data.append(val)

        if not self.min or val <= self.min[-1]:
            self.min.append(val)
        

    def pop(self):
        """
        :rtype: None
        """
        if self.data.pop() == self.min[-1]:
            self.min.pop()
        

    def top(self):
        """
        :rtype: int
        """
        return self.data[-1]
        

    def getMin(self):
        """
        :rtype: int
        """
        return self.min[-1]
```
