## 题意

给定整数数组 `nums`，找出所有和为 0 且不重复的三元组。返回结果不能含重复三元组。

## 思路

先想哈希表套两数之和，发现三数去重特别恶心，时间还会升到 O(n³)。

换思路：排序后固定一个数，剩下两个用双指针往中间夹。排序后去重只需要跳过相邻重复值，指针移动也有单调性。时间降到 O(n²)。

去重要注意：必须在"已经用过该值"之后跳，不能提前跳，否则漏解。

## 代码

```python
class Solution(object):
    def hasCycle(self, head):
        """
        :type head: ListNode
        :rtype: bool
        """
        fast = slow = head
        while fast and fast.next:
            slow = slow.next
            fast = fast.next.next
            if slow == fast:
                return True
        return False
```
